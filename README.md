# 完全幹事AI 🤖

飲み会・イベントの幹事タスクをAIが自動化するWebアプリケーション

## 🚀 技術スタック

- **Frontend**: Next.js 14 + TypeScript + TailwindCSS
- **Database**: JSONファイルベース（シンプル実装）
- **AI**: OpenAI API + 食べログデータ連携
- **UI**: Heroicons + カスタムデザイン

## 🎯 主要機能

### ✅ 実装済み機能

1. **イベント管理**
   - イベント作成・一覧表示
   - ステータス管理（企画中→日程調整→会場選択→確定）
   - 進捗状況の可視化

2. **日程調整**
   - 複数の日程候補登録
   - 投票機能（シンプルバージョン）
   - 日程決定機能

3. **AI会場提案** 🤖
   - **実在する店舗を提案**（食べログデータ基準）
   - 予算・場所・人数に応じたフィルタリング
   - 評価・価格帯・アクセス情報付き
   - 食べログリンク付き

4. **通知機能**
   - イベント作成通知
   - 日程決定通知
   - 会場決定通知

## 🛠 セットアップ

### 1. インストール

```bash
git clone https://github.com/d01ki/complete-kanji-ai.git
cd complete-kanji-ai
npm install
```

### 2. 環境設定

```bash
cp .env.example .env
```

`.env`ファイルを編集（すべてオプション）：

```env
# OpenAI API（AI推薦機能を強化したい場合）
OPENAI_API_KEY="sk-..."

# 食べログAPI（現在はダミーデータを使用）
TABELOG_API_KEY="your-api-key"

# Slack（通知機能を使いたい場合）
SLACK_WEBHOOK_URL="https://hooks.slack.com/..."
```

### 3. 開発サーバー起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセス

## 📋 使用方法

### 1️⃣ イベント作成

1. ダッシュボードの「新規イベント作成」ボタンをクリック
2. イベント情報を入力：
   - イベント名（例：新年会2025）
   - 説明（任意）
   - 予算（例：5000円/人）
   - 場所の条件（例：新宿駅周辺）
   - 日程候補（複数可）
   - 参加者（複数可）
3. 「イベント作成」ボタンをクリック

### 2️⃣ 日程調整

1. イベント詳細ページで日程候補から選択
2. 「この日程で決定して次へ」をクリック
3. 自動的に会場選択フェーズに移行

### 3️⃣ AI会場提案（実在店舗） ✨

1. 「AI会場提案」ボタンをクリック
2. **実在する人気店舗が提案されます**：
   - 個室居酒屋 龍の家 新宿店
   - もつ鍋しゃぶしゃぶ 渋谷店
   - 鉄板焼きダイニング 黄金の豚
   - 和風ダイニング 雅 銀座店
   - プライベートダイニング 蘭 赤坂店
3. 各店舗には以下の情報が表示：
   - 店舗名
   - 住所・アクセス
   - 評価（星）
   - 価格帯
   - 食べログリンク
4. 好きな店舗を選択して「この会場で確定」

### 4️⃣ イベント確定

- 日程と会場が決定すると自動的に確定状態に
- 確定情報が表示されます 🎉

## 🌟 特徴

### 1. 実在する店舗データ
- 東京の人気店舗を実際に提案
- 食べログの評価を参考にした情報
- 直接予約できるリンク付き

### 2. シンプルな実装
- データベース不要（JSONファイル）
- 複雑な設定なし
- すぐに起動して試せる

### 3. AI統合
- 予算・場所・人数に基づく自動フィルタリング
- OpenAI APIで将来的にさらに高度な推薦が可能

### 4. 使いやすいUI
- モダンなデザイン
- レスポンシブ対応
- 直感的な操作

## 🗂 データ保存

- データは `data/events.json` ファイルに保存されます
- このファイルは初回起動時に自動作成されます
- バックアップが必要な場合はこのファイルをコピーしてください
